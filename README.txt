Pour essayer cette démo, vous devez remplacer le composant BSPK.4DBase, vide, du projet par la dernière version téléchargée sur notre site : https://bspoke.fr/ressources

En prévision de votre première visite, il faudra modifier votre fichier host pour ajouter les entrées suivantes : 
127.0.0.1	demo1.localhost
127.0.0.1	safari.localhost
127.0.0.1	commande.localhost
127.0.0.1	myproducts.localhost

Une fois tout cela fait, lancer la base 4D démo et ouvrir sur votre navigateur sur l'un des domaines disponibles.


To try this demo, you need to replace the project’s empty BSPK.4DBase component with the latest version downloaded from our site : https://bspoke.fr/en/resources

In anticipation of your first visit, you will need to modify your host file to add the following entries: 
127.0.0.1	demo1.localhost
127.0.0.1	safari.localhost
127.0.0.1	commande.localhost
127.0.0.1	myproducts.localhost

Once all this is done, launch the 4D demo database and open on your browser on one of the available domains.
